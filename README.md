# EvoTwin AI

Offline-first, self-evolving personal AI for Android with Gemma LiteRT, memory system, and phone automation.

## Features
- Digital clone of the user
- Offline AI (Gemma LiteRT)
- Memory system (SQLite + FAISS-like)
- Self-improvement engine
- Voice output and cloning
- Phone automation

## Setup
1. Clone the repo
2. Open in Android Studio
3. Place `gemma-4-E2B-it.litertlm` in `app/src/main/assets/`
4. Build APK
